  User Actions in Setup Mode:

  Step 1: Connect to WiFi AP
  - Connect your phone/laptop to WiFi: ModbusIoT-Config
  - Password: config123

  Step 2: Open Web Interface
  - Open browser: http://192.168.4.1
  - You'll see the Fluxgen configuration interface

  Step 3: Configure Network
  - WiFi Mode: Enter your WiFi SSID and password
  - OR SIM Mode: Enter APN details for 4G module

  Step 4: Configure Sensors
  - Add Modbus sensors (Panda USM, Flow meters, etc.)
  - Set slave IDs, register addresses, data types

  Step 5: Configure Azure IoT Hub
  - Enter Device ID (e.g., gateway-01)
  - Enter Device Key (from Azure portal)
  - Set telemetry interval (e.g., 60 seconds)

  Step 6: Start Operation
  - Click "Start Operation Mode" button
  - System saves all configuration to NVS flash
  - ESP32 automatically restarts
